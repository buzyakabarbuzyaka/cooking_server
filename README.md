# Требования
* docker
* docker-compose

# Использование
```
# консоль 1
# рабочая директория <ПУТЬ ДО ПАПКИ>/cooking_server
docker-compose up
```
После этой команды должна появится папка __input__
```
# консоль 2
# рабочая директория <ПУТЬ ДО ПАПКИ>/cooking_server
chmod -R 777 input
```
Теперь в эту папку можно закинуть файл __data.json__
